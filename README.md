# Quadcopter Drone 🚁

## Overview
A fully assembled quadcopter drone built from scratch using individual 
hardware components. The project covers component selection, assembly, 
ESC calibration, and basic flight testing.

## Components Used
| Component | Details |
|---|---|
| Frame | F450 Quadcopter Frame |
| Motors | Brushless DC Motors |
| ESC | Electronic Speed Controllers |
| Flight Controller | FC with gyroscope/accelerometer |
| Battery | LiPo Battery |
| Transmitter/Receiver | RC Remote |

## What Was Done
- Assembled all components onto the F450 frame
- Connected and configured ESCs with brushless motors
- Calibrated flight controller for stable hovering
- Performed test flights for directional control

## Learnings
- Basics of drone aerodynamics and thrust balance
- ESC calibration and motor direction configuration
- Flight controller PID tuning fundamentals

## Status
✅ Successfully hovered and achieved basic directional control
